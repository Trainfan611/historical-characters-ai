# ✅ Улучшения входа и смены аккаунта

## 🎯 Что было сделано

### 1. Автоматический редирект на страницу генерации

**После успешного входа через Telegram:**
- ✅ Пользователь автоматически перенаправляется на `/generate`
- ✅ Не нужно вручную переходить на страницу генерации

**Реализация:**
- Обновлен `components/auth/TelegramLogin.tsx`
- После успешного `signIn` выполняется `router.push('/generate')`

### 2. Возможность смены аккаунта Telegram

**Добавлено:**
- ✅ Кнопка "Сменить аккаунт Telegram" на странице профиля
- ✅ Кнопка "Выйти" в Header теперь перенаправляет на `/login`
- ✅ После выхода можно войти другим аккаунтом Telegram

**Где найти:**
1. **На странице профиля** (`/profile`):
   - Кнопка "Сменить аккаунт Telegram" в карточке пользователя
   
2. **В Header** (на всех страницах):
   - Кнопка "Выйти" в правом верхнем углу

### 3. Улучшенная логика страницы входа

**Если пользователь уже авторизован:**
- ✅ При переходе на `/login` автоматически перенаправляет на `/generate`
- ✅ Не нужно вручную выходить, чтобы войти другим аккаунтом

## 🔄 Как это работает

### Сценарий 1: Первый вход

1. Пользователь открывает `/login`
2. Нажимает "Войти через Telegram"
3. Авторизуется через Telegram виджет
4. **Автоматически перенаправляется на `/generate`**

### Сценарий 2: Смена аккаунта

**Вариант А: Через профиль**
1. Пользователь открывает `/profile`
2. Нажимает "Сменить аккаунт Telegram"
3. Перенаправляется на `/login`
4. Может войти другим аккаунтом Telegram

**Вариант Б: Через Header**
1. Пользователь нажимает "Выйти" в Header
2. Перенаправляется на `/login`
3. Может войти другим аккаунтом Telegram

### Сценарий 3: Повторный вход

1. Пользователь уже авторизован
2. Открывает `/login`
3. **Автоматически перенаправляется на `/generate`**
