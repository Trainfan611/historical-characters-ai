# Краткое описание проекта для программы "Маяк"

## 🎯 Название проекта
**Historical Characters AI** — платформа для генерации реалистичных изображений исторических личностей с помощью AI

## 📝 Краткое описание (1-2 предложения)
Веб-платформа, которая позволяет создавать высококачественные изображения любых исторических персонажей с помощью искусственного интеллекта. Уникальность проекта — в автоматическом расширении базы данных через поиск в интернете, что позволяет работать с неограниченным количеством исторических личностей.

## 💡 Проблема
Недоступность качественных визуализаций исторических личностей для образовательных целей, высокая стоимость создания профессиональных иллюстраций, ограниченность существующих баз данных.

## ✅ Решение
Платформа с AI-генерацией изображений, динамической базой данных (автоматический поиск через интернет), интеграцией Telegram для удобной авторизации и монетизации.

## 🚀 Уникальность
1. **Динамическая база** — автоматический поиск информации о любой исторической личности через Perplexity API
2. **Самообучающаяся система** — база расширяется с каждым запросом
3. **Современный стек** — Next.js 14, Flux AI, интеграция нескольких AI-сервисов

## 💼 Бизнес-модель
- Freemium модель с премиум-подписками
- B2B — API для образовательных платформ и медиа
- Партнерства с образовательными каналами и музеями

## 📊 Статус
✅ Полнофункциональная платформа реализована и работает  
🌐 Деплой: [history-character.up.railway.app](https://history-character.up.railway.app)  
🛠️ Технологии: Next.js 14, TypeScript, PostgreSQL, Flux AI, Perplexity API

## 🎯 Планы
- Масштабирование пользовательской базы
- Внедрение монетизации (премиум-подписки, API)
- Партнерства с образовательными платформами
- Развитие B2B-направления

Студент 3 курса направления "Менеджмент. Предпринимательство и управление бизнесом". Развиваю собственный бизнес в e-commerce и нейросетевые проекты. Имею практический опыт в full-stack разработке и работе с AI-технологиями.
